# 民第 154 条

</div>


**[[法条/法条细则/民法典/第一百五十四条\|第一百五十四条]]**　行为人与相对人恶意串通，损害他人合法权益的民事法律行为无效。 

</div></div>


恶意串通：在买卖当中双方以损害他人利益为目的，弄虚作假的违法行为 #背诵/概念 
- 损害第三人利益
- 与通谋虚伪表示对比
# 法律后果：无效
