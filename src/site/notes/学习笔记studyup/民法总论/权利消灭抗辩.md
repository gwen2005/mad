# 法效果
- 抗辩的主张的效果：
	- 不产生实体法层面上的效果：权利的不发生或者消灭是既定事实，并未因被请求人主张或者不主张抗辩而改变
	- 在程序法层面上避免法官作出不符合真实法律状况的裁判>>>>判令被告履行已经消灭的债务
